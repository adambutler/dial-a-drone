# Dial-a-Drone

## Install

```
$ git clone git@github.com:adambutler/dial-a-drone.git
$ cd dial-a-drone
$ npm install
$ cp .env.example .env
```

## Usage

```
$ coffee drone.coffee
$ ruby dialadrone.rb
```
